# blue_pill
## Description
Matrix-esque falling symbols with c++ and sfml.

Language: c++ & glsl

Dependencies: sfml

## Installation
### Install prerequisites
* gcc
* sfml
Example (Arch):
```
$ sudo pacman -S sfml
$ sudo pacman -S gcc
```

### Build
```
$ git clone https://github.com/DennisLovkvist/blue_pill.git
$ cd blue_pill
$ ./build.sh
```
### Run
```
$ ./blue_pill
```
